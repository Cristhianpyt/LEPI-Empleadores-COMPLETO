## 🗂️ Proyectos por Componente LEPI

### 🔵 L — Liderazgo Docente
**Problema:** Medir y predecir el perfil de liderazgo de 200+ docentes.  
**Solución:** Índice compuesto ILD + K-Means + Gradient Boosting.  
**Resultado:** R² = 0.81, 4 perfiles identificados con 94% de pureza.

### 🟡 P — Praxeológico (Ver → Juzgar → Actuar → Devolver)
**Problema:** Estructurar el análisis siguiendo el ciclo reflexivo de Juliao.  
**Solución:** Pipeline de datos en 4 fases alineado al marco praxeológico.  
**Resultado:** Correlación significativa r = 0.73 liderazgo → rendimiento.

### 🟢 E — Educativo
**Problema:** Detectar docentes en riesgo de bajo desempeño antes de evaluaciones formales.  
**Solución:** Random Forest con SMOTE + sistema de alertas.  
**Resultado:** AUC = 0.84, reducción del 28% en baja evaluación no anticipada.

### 🔴 I — Innovador
**Problema:** Medir el impacto de la innovación pedagógica sobre el rendimiento estudiantil.  
**Solución:** Regresión con variables de innovación + evaluación antes/después.  
**Resultado:** Δ ILEPI = +0.82 puntos, significativo (p < 0.001).
